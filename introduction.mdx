## Welcome to TeamSpeak 6 Server

Welcome to the Beta release of TeamSpeak 6 Server! Experience the next generation of voice communication with TeamSpeak 6 - trusted for nearly 25 years and now better than ever.

<Info>
This is a **Beta release**. The main objective is to collect diverse feedback and identify potential issues before the stable version is launched. Your feedback is essential to help us shape the future of TeamSpeak.
</Info>

## Why TeamSpeak?

TeamSpeak offers the ideal voice communication solution for:

- **Gaming communities** - Crystal clear voice with ultra-low latency
- **Education and training** - Reliable communication for virtual classrooms
- **Internal business communication** - Secure and professional voice infrastructure
- **Friends and family** - Stay connected with those who matter most

Our primary focus is delivering a solution that is easy to use, with high security standards, excellent voice quality, and low system and bandwidth usage.

## Beta License Information

<Note>
The server includes a **32-slot Beta License** that will be renewed every two months throughout the beta/evaluation period, until licensing and pricing are finalized.
</Note>

**Important limitations:**

- TeamSpeak 3 Server licenses are **not compatible** with TeamSpeak 6 Servers
- There is currently **no migration path** available between the two versions
- It is **not yet possible** to obtain or upgrade to a larger license for TeamSpeak 6

We appreciate your patience and understanding as we continue to develop solutions to better support your needs.

## Deployment Options

Choose the deployment method that works best for you:

### Docker (Recommended)

The easiest way to run TeamSpeak 6 Server. Docker provides an isolated, portable, and easily manageable environment, making setup and maintenance straightforward.

### Native Binaries

Run the server directly on Linux or Windows for more control over your environment. Perfect for advanced users who want direct access to the server process.

### Managed Hosting

Don't want to self-host? Rent official TeamSpeak 6 Servers directly through [TeamSpeak Communities](https://www.myteamspeak.com/communities) for a hassle-free experience.

## Key Features

### Flexible Configuration

Configure your server in three main ways:

1. **Command-Line Arguments** - Pass options directly when starting the server
2. **Environment Variables** - Set variables before starting (ideal for Docker)
3. **YAML Configuration File** - Persistent configuration for production deployments

### Database Support

- **SQLite** - Zero-configuration database, perfect for getting started
- **MariaDB** - Production-grade database for high-performance deployments

### Server Query APIs

- **HTTP Query** - RESTful API for server management
- **SSH Query** - Secure shell access for advanced administration
- **HTTPS Query** - Encrypted web-based query interface
